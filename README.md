# Employee Skills CV App

SAP CAP application skeleton for managing employee skills, skill catalog governance, CV uploads, skill extraction, and generated CV documents.

Current scope:

- CAP project structure
- CDS domain model in `db/schema.cds`
- No services or UI yet

Next likely steps:

1. Add CAP services in `srv/`.
2. Add seed data in `db/data/`.
3. Install dependencies with `npm install`.
4. Run locally with `npm run watch`.
